Simply a port of https://github.com/rui314/minilisp to Zandronum with the GDCC compiler. To try it out, write a chat message with a `$` at the start. The rest of the message will be interpreted as a Lisp expression.

The language is largely identical to that in the original repository, with the addition of `*`, `/`, `>`, `<=`, `>=` functions. You can probably guess what they do.
